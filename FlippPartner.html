Ensuring the Right Flyer Reaches the Right User

Multi-Layered Targeting

We use a waterfall logic to ensure maximum accuracy and reach.

  1. DMP Cookie Match: The primary method, linking directly to your loyalty data.
  2. Flipp User Profile: A fallback using Flipp's first-party data (shopping lists, clicks).
  3. Default Experience: Unmatched users see the standard flyer. No one is left behind.

The Matching Process

1

User visits with DMP cookie.

2

Flipp's system detects the cookie.

3

User is mapped to a segment.

4

Personalized flyer is served.
